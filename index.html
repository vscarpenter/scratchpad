    About Scratchpad

    Scratchpad keeps everything you write in this browser only, using IndexedDB. Nothing is uploaded, synced, or transmitted. There are no accounts, trackers, or analytics.

    Clearing site data (or using private browsing) will erase your notes. Back up regularly with Export, and restore with Import.

    Keyboard shortcuts

    • ⌘/Ctrl + N — New note
    • ⌘/Ctrl + S — Save (in edit mode)
    • ⌘/Ctrl + K or / — Focus search
    • Esc — Clear search, or exit edit mode
